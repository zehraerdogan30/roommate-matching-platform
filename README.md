# Roommate Matching Platform

A Flutter-based mobile application designed to help users find compatible roommates based on lifestyle preferences, habits, and personal compatibility.

## Features

- User authentication (Firebase)
- Profile creation and editing
- Lifestyle questionnaire
- Compatibility-based roommate matching concept
- Roommate request system
- Firestore database integration

## Tech Stack

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
